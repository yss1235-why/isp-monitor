This monitoring dashboard has the following limitations:

  • 15-minute sync delays: Data is synced from UptimeRobot every 15 minutes via GitHub Actions, not real-time
  • GitHub Actions dependency: Data sync reliability depends on GitHub's free tier availability and may experience delays
  • UptimeRobot service dependency: Monitoring accuracy depends on UptimeRobot's service availability and infrastructure
  • Free tier limitations: UptimeRobot free tier has 50 monitor limit and 5-minute check intervals
  • Public IP monitoring: Monitors ISP gateway/DNS servers, not actual broadband infrastructure or routing quality
  • Limited scope: Only tests basic connectivity, not bandwidth, DNS resolution, or actual internet browsing experience
  • Data freshness: Historical calculations depend on available data - newer ISPs may show limited uptime history
  • Geographic variance: UptimeRobot monitors from their locations, which may not reflect your local network conditions
