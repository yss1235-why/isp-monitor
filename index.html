This monitoring tool displays UptimeRobot data and has the following limitations:

  • 5-minute intervals: UptimeRobot checks every 5 minutes (free tier)
  • External perspective: Tests from UptimeRobot's global network, not your local network
  • Limited free tier: 50 monitors maximum, basic features only
  • Geographic variance: Results may differ from your local network experience
  • Monitor types: Limited to HTTP/HTTPS, Ping, Port, and Keyword monitoring
  • Data sync delay: Dashboard updates every 30 minutes via GitHub Actions
  • No bandwidth testing: Only tests connectivity and response time
  • Single data source: Relies entirely on UptimeRobot service availability

Use this tool for general monitoring trends only, not for critical decisions or SLA monitoring.

For comprehensive ISP monitoring, consider multiple monitoring services and local network testing.
